# Drivex

A React + Vite landing page for Drivex, a lightweight launch and analytics workspace.

## Features

- Modern React UI built with Vite for fast feedback.
- Responsive landing page with hero section, feature cards, and external docs links.
- Clean, production-ready CSS and page structure.

## Run locally

```bash
npm install
npm run dev
```

Then open the local Vite development URL shown in the terminal.

## Build

```bash
npm run build
```
